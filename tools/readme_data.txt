The static data that drives this application comes from ships.csv, and can be converted to a suitable PHP array by make_arrays.py.

If you wish to update the data:

1. Make your modifications to ships.csv
2. Run make_arrays.py
3. Take the new array from ships.txt and replace the existing one in static_data.php

Note that the original static data was sourced from the EVE University wiki, and enhanced by the author of DScan Reporter application.
